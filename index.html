Sobre mim

Olá sou Emerson, tenho 27 nos, sou formado em Análise e Desenvolvimento de Sistemas pela UNINOVE, comecei na área de programação a um pouco mais de um ano, inicialmente como estágiario em Cobol, posteriormente migrei para estágiario em Java, onde participei do projeto VEIC, na parte de prototipagem utilizando a linguagem Java e framework Java Server Faces com a arquiterura Bradesco AWB 3.0.
Atualmente estou alocado no cliente Bradesco, no projeto BRAIN, um projeto piloto que consiste em uma API Rest, utilizando a linguagem java, metodologia ágil, e esteira de entrega contínua.
